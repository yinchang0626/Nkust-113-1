臺灣豬證明標章商家資料管理系統



專案名稱
臺灣豬證明標章商家資料管理系統

專案簡介
本專案基於政府開放資料設計與實現了一個完整的資料庫與網頁系統，主要功能為管理和展示臺灣豬證明標章商家資料，並提供搜尋、篩選、排序、視覺化數據等互動功能。

資料集來源
資料來源為臺灣政府開放資料平台： 臺灣豬證明標章商家資料集。
https://data.gov.tw/dataset/136370

系統架構
前端：採用 HTML、CSS、JavaScript 搭建，提供使用者友好的操作介面及互動功能。
後端：使用 Node.js（Express 框架）實現 RESTful API，處理資料庫操作與邏輯。
資料庫：基於 AWS RDS MySQL 儲存與管理商家數據。
功能特性
資料庫功能：
設計商家數據表，包含基本資訊如名稱、地址、營業時間等。
支援數據的新增、修改、刪除、查詢操作。

前端功能：

搜尋商家名稱或地址。
根據營業時間排序或區域篩選商家資料。
顯示商家資料表及數據的動態圖表（圓餅圖、柱狀圖、折線圖）。

後端功能：

提供資料操作的 API，包括新增、修改、刪除與查詢。
使用 MySQL 儲存數據並處理篩選及排序邏輯。



**使用方法**

**資料庫有設定過了可以直接跳到執行程式**

設定資料庫

連接 AWS RDS MySQL：
使用以下指令登入資料庫：
複製程式碼到終端機執行
mysql -h database-1.c54qu48sokg2.ap-northeast-3.rds.amazonaws.com -P 3306 -u garden -p

建立資料表與導入數據：

使用 SQL指令.txt 中的指令建立資料表並匯入 JSON 數據​SQL指令。
透過 input_sql.py 將資料自動導入​input_sql。

**執行程式**

把壓縮檔解壓縮後
node {路徑}/server.js

伺服器啟動後，訪問 http://localhost:3000 進入網頁。

進入首頁，瀏覽所有商家資料。
使用搜尋框輸入關鍵字查詢商家。
透過篩選選單選擇區域，或按營業時間排序商家。
點擊圖表按鈕生成視覺化數據。
