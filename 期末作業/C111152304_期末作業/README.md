這是一個使用 ASP.NET Core MVC 建立的 Web 應用程式，允許用戶管理和查看股票資料。它支援篩選、排序以及股票資料的 CRUD 操作（建立、讀取、更新、刪除）。

功能
• 查看股票資料：列出所有股票資料，並可根據證券代號、證券名稱或成交金額進行篩選與排序。
• 新增股票資料：將新的股票紀錄新增至資料庫。(需改善)
• 編輯股票資料：修改現有的股票紀錄。(需改善)
• 刪除股票資料：從資料庫中移除股票紀錄。(需改善)
• 搜尋功能：根據證券代號或名稱篩選股票資料。

.NET 6 或更高版本
已配置資料庫連接字串，預設使用 SQL Server Object Explorer
支援 ASP.NET Core 的 IDE，如 Visual Studio
